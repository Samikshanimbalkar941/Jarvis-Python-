# AI Voice Assistant (Jarvis)

This is a Python based voice assistant similar to Jarvis.

## Features
- Speech recognition
- Voice responses
- Wikipedia search
- Open websites
- AI responses

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- OpenAI API

## Installation

Clone this repository

git clone https://github.com/yourusername/AI-Voice-Assistant-Jarvis.git

Install requirements

pip install -r requirements.txt

Run the program

python main.py
